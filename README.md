# TIMES NEWS website(타임즈 뉴스 웹사이트)

![]("img/demo.png")
![]("img/demo-mobile.png")

## 사용한 기술

- HTML,CSS,Javascript,API(https://newsapi.org/docs)
- API 활용해 이미지, 제목, 내용, 소스, 날짜 구성
- 카테고리에 나누어진 주제로 뉴스 검색 가능
- 검색창 활용해 키워드 검색으로 뉴스 검색 가능
- 반응형 웹페이지로 모바일 버전에서는 카테고리가 사이드 메뉴바로 제공
- 페이지네이션 사용
